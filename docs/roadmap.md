# Roadmap

<p class="page-intro">This page presents delivery priorities by horizon to support planning, governance, and sequencing decisions.</p>

## Delivery Principle
Priorities are sequenced to complete functional integration first, establish quality gates second, and expand strategic capabilities third.

## Near-Term Priorities
1. Complete frontend integration for all implemented backend modules.
2. Add baseline automated tests for backend and frontend critical paths.
3. Harden deployment configuration and operational readiness.

## Mid-Term Enhancements
1. Introduce richer dashboard analytics and trend indicators.
2. Improve budgeting insights with variance and forecasting support.
3. Expand role and permission capabilities as collaboration requirements evolve.

## Long-Term Direction
1. Improve interoperability with external financial data providers.
2. Extend reporting exports and audit-friendly history views.
3. Evaluate mobile-first clients after web parity and quality gates are stable.
